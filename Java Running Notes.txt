WHY WE NEED TO LEARN JVM?

    - WITHOUT JVM WE CAN NOT EXECUTE THE JAVA PROGRAM.
	- FOR COMPILATION WE NEED JDK(JAVA DEVELOPMENT KIT).
	- JDK = JRE + JVM. OR
	- JDK = JRE + DEVELOPMENT TOOLS(JAVAC, JAVA, JRE, JCONSOLE, JDB.....).
	- COMPILATION CAN BE DONE BY COMPILER.
	- EXECUTION CAN BE DONE BY JVM ALONG WITH INTERPRETER. 
	- JVM CONVERTS BYTE CODE TO NATIVE CODE. 
	
	
DIFFERENCE BETWEEN JDK, JRE AND JVM?
JDK IS A OPEN SOURCE(FREE ALONG WITH ANY OTHER DEVELOPMENTS ALSO POSSIBLE) SOFTWATE TO DOWNLOAD AND INSTALL IN YOUR SYSTEM TO COMPILE AND EXECUTE JAVA PROGRAMS.
JRE = LIBRARY CLASSES +JVM TO PROVIDE THE ENVIRONMENT FOR THE PRE-DEFINED CLASSES.
JVM IS AN INTERNAL SYSTEM TO CONVERT BYTECODE TO NATIVECODE AND EXECUTE A JAVA CODE.
	
EXECUTION PROCESS:

.JAVA --->  SOURCE CODE ---AFTER COMPILATION--> BYTE CODE
.CLASS --> BYTE CODE --- AFTER EXECUTION --> NATIVE CODE --> OS --> OUTPUT.
